# The Primes package

Enabling computations with prime numbers in python since March 2016.

## Set-up
From bash:
```
$ source setup/setup.sh
```
[t]csh:
```
$ source setup/setup.csh
```

## Testing
```
$ nosetests
....
----------------------------------------------------------------------
Ran 4 tests in 0.003s

OK
```

## Demo
```
>>> from desc.primes import Primes
>>> Primes.is_prime(17)
True
>>> Primes.is_prime(20)
False
```
